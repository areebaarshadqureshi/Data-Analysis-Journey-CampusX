## ▶️ How to Run Notebooks

### Option 1 — Google Colab (Recommended)
1. Open any `.ipynb` file in this repo
2. Click the **Open in Colab** button at the top
3. Run the cells directly in your browser — no installation needed!

### Option 2 — Jupyter Notebook (Local)
1. Clone this repository:
```bash
git clone https://github.com/areebaarshadqureshi/Data-Analysis-Journey-CampusX.git
```
2. Install required libraries:
```bash
pip install pandas numpy matplotlib seaborn
```
3. Open Jupyter Notebook:
```bash
jupyter notebook
```
4. Navigate to any notebook and run it!
